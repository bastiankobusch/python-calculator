# Python Calculator

A beginner Python project that performs basic calculations, exception handling and language selection between EN and DE.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Rounds digits to 3 decimal places
* Expeption handling
* Language selection: EN or DE
* Simple command-line interface

## Purpose

This project was created to further learn and practice applying basic Python skills like variables, using basic arithmetic operators, conditions, loops, exception handling, using UTF-8 to add more letters, importing yaml-files and user input and ouput.
